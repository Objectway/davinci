Sass Grid Generator based on CSS3 Flex-box

It's developed thinking about modern web applications and its first purpose is to make easier to manage responsive layout.

Customizable

Using the Sass version you can personalize SuperGiGi to define the number of columns, breakpoints, name of grid components, etc.

Preprocessors

It's developed in Sass but if you prefer you can use a compiled version.

Mobile first

You can build for small devices first. Then, as devices get larger and larger, SuperGiGi easily scale your applications for a complete responsive design.
